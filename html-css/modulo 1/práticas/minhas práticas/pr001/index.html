Símbolos

No HTML, para adicionar símbolos especiais, usamos as chamadas entidades HTML.

Elas seguem geralmente esse formato: &nome_da_entidade ou &#número

Emojis

No HTML, para adicionar emojis podemos copiar e colar os emoji no código ou usamos hexadecimais.

As hexadecimais geralmente seguem geralmente esse formato: &#x...
