O objetivo deste código é criar uma página HTML completa, com base nos conhecimentos adquiridos nos exercícios 1 ao 10 do Módulo 1.

Essa página tem como finalidade explicar, de forma simples, como utilizar símbolos e emojis em HTML. Além disso, ela deve conter listas. E referências para documentações oficiais ou enciclopédias confiáveis externas.
